# simulation

## 目录定位
仿真层单元测试目录。

## 职责范围
- 仿真输入层测试
- 查询与状态测试
- 系统执行流测试

## 允许放入
- simulation 纯逻辑测试
- 测试上下文与仿真专用轻量辅助

## 禁止放入
- 网络集成链路测试
- 场景驱动型冒烟测试
- 非仿真业务测试

## 对外依赖
- 可依赖 `res://gameplay/simulation/` 与 `res://tests/helpers/`
- 不依赖前台/战斗正式场景

## 维护约束
- 保持 deterministic 取向
- 命名使用通用测试名
- 不继续使用原型阶段命名
