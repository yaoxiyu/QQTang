# scenes

## 目录定位
正式场景与 sandbox 场景目录。

## 职责范围
- 前台场景
- 战斗场景
- 网络启动场景
- sandbox 场景承载

## 允许放入
- 正式 `.tscn`
- 与场景绑定的控制器
- 场景契约相关文档引用

## 禁止放入
- 阶段性命名散落
- 无文档的临时场景
- 与场景无关的脚本堆积

## 对外依赖
- 可依赖 app/gameplay/network/presentation 正式路径
- 不承担资源真相定义职责

## 维护约束
- 正式入口与 sandbox 明确分离
- 场景重命名需同步脚本路径
- 不再继续按 phase 命名
