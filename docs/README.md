# docs

本目录只保留需要随代码维护的平台 API 与内部协议契约。

## 当前目录

- `current_source_of_truth.md`：最小文档入口。
- `platform_auth/`：账号、资料、票据、错误码契约。
- `platform_game/`：匹配、职业、结算、Battle 分配与内部 Game Service 契约。
- `platform_room/`：Room Service runtime 契约。
- `technical_debt.md`：随代码维护的工程技术债台账。

设计文档、阶段文档、验收记录和运行时说明已迁移到外部归档。工程技术债允许在仓库内维护，以便和代码变更、测试闭环一起演进。
